# Python DevContainer with VS Code Extensions

This project provides a ready-to-use development environment in Visual Studio Code using GitHub Codespaces or local devcontainers. It includes Python 3, the `uv` package manager, and a curated set of VS Code extensions for Python, Jupyter, PDF, SQLite, CSV, and Jekyll workflows.

## Features

- Python 3 with `uv` for fast dependency management
- Pre-installed VS Code extensions:
  - GitHub Theme
  - Python & Pylance
  - Windows AI Studio
  - PDF Viewer
  - SQLite Viewer
  - Jupyter
  - Rainbow CSV
  - Jekyll Post
- Automatic virtual environment setup with `uv`

## Usage

1. Open this folder in VS Code and reopen in container when prompted.
2. The devcontainer will build and set up the environment automatically.
3. Start coding in Python, Jupyter, or use the included tools as needed.

---

For more details, see `.devcontainer/devcontainer.json` and `Dockerfile`.
